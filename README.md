# PersonalBlogApplication
# 📝 Blogging Platform

This project is a web-based blogging platform developed to allow users to create, manage, and interact with blogs in a simple and efficient way. The platform provides features like user authentication, blog creation using Markdown, commenting system, and basic chatbot assistance.

Users can register and log in securely, create and edit blog posts, and view content shared by others. The chatbot helps users with basic guidance while using the platform.

The backend of the project is built using Node.js and Express.js, handling all server-side logic and APIs. Data is managed efficiently to ensure smooth performance.

This project is developed for academic purposes and focuses on implementing core backend functionalities and real-world concepts like REST APIs, authentication, and data handling.

---

## 🚀 Features
- User Authentication (Login/Register)
- Create, Edit, Delete Blogs
- Markdown Support for Writing Blogs
- Comment System
- Basic Chatbot Assistance
- REST API Integration

---

## 🛠️ Tech Stack
- Node.js
- Express.js
- JavaScript
- JSON (for data storage)

---

## 📌 Note
This project is a basic implementation and does not include advanced features like real-time collaboration, AI recommendations, or mobile support.
